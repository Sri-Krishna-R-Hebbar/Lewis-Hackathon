Instructions

Word Search link

Click on the link and complete the word search.

  1. After finishing the word search, submit a screenshot of it on the website.
  2. Enter your HTML code in the HTML slot.
  3. Enter your CSS code in the CSS slot.
  4. Enter your JavaScript code in the JavaScript slot.
  5. Click the “Run” button.
  6. Please wait for 5-10 seconds for the website compiler to be ready
  7. If your word search is correct and all keywords are found, your website will be created instantly.
